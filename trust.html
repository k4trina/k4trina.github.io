Security, privacy, and data protection at Bidblox

Bidblox is built with enterprise-grade security practices to ensure customer data and project information remain protected, private, and fully under customer control.

Security Overview

  • ✓ Data encrypted in transit (TLS 1.2+)
  • ✓ Data encrypted at rest
  • ✓ Role-based access control (RBAC)
  • ✓ Audit logging across systems
  • ✓ Continuous monitoring and security alerts

Customer Data & AI Policy

Customer data remains fully owned by the customer. Bidblox is designed to ensure complete confidentiality and control over all uploaded project information.

  • ✓ We do not sell customer data
  • ✓ We do not share customer data with third parties except to provide the service
  • ✓ Customer data is not used to train public AI models
  • ✓ All project files remain private to the customer environment

Infrastructure & Hosting

Bidblox runs on Google Cloud Platform (GCP), leveraging secure, scalable, enterprise-grade cloud infrastructure.

  • ✓ Hosted on Google Cloud Platform (GCP)
  • ✓ Data stored in secure, access-controlled cloud environments
  • ✓ Encryption in transit and at rest
  • ✓ Role-based access controls (RBAC)
  • ✓ Continuous monitoring and automated backups
  • ✓ Built-in cloud security and DDoS protection

Access Controls

  • ✓ Role-based permissions across the platform
  • ✓ Restricted production system access
  • ✓ Secure authentication mechanisms
  • ✓ Activity logging for sensitive actions
